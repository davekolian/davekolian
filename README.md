## 🧠 What I'm Building

I've spent the past year deep in the weeds designing and shipping systems like:

- **Multi-agent LLM architectures** - hierarchical routing, semantic search, and tool-calling agents built with LangGraph
- **Computer vision pipelines** - YOLO object detection fused with Vision-Language Models for real-world hazard analysis
- **Deterministic conversational frameworks** - hallucination-resistant, JSON-defined state machines for safety-critical user flows
- **AI infrastructure** - MCP servers, cost governance systems, and multi-provider LLM orchestration

The part I love most? Watching all of it hold up in front of real users, not just in a demo. I'm obsessed with building things that actually ship. Give me the model, the backend, and the interface - I want to own the whole thing.
